¿Qué es XML?

XML es el acrónimo de Extensible Markup Language, es decir, es un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos.

XML se derivó de SGML (Standard Generalized Markup Language), surge como evolución de otros lenguajes de marcado con el objetivo de mejorar la interoperabilidad y la organización de datos. Su desarrollo fue impulsado por el W3C con la finalidad de simplificar el intercambio de informaciones estructuradas en la web y otros sistemas.

Ejemplo de uso:

<?xml version="1.0" encoding="UTF-8"?>
<biblioteca>
    <libro categoria="ficcion">
        <titulo>Don Quijote de la Mancha</titulo>
        <autor>Miguel de Cervantes</autor>
        <año>1605</año>
    </libro>
    <libro categoria="programacion">
        <titulo>Aprendiendo XML</titulo>
        <autor>Juan Pérez</autor>
        <año>2023</año>
    </libro>
</biblioteca>

Principales usos:

  • Intercambio de datos: Muy usado en integración entre aplicaciones, especialmente cuando están escritas en distintos lenguajes o corren en diferentes plataformas.
  • Configuración: Muchos programas y frameworks usan archivos XML para guardar configuraciones.
  • Almacenamiento: Similar a una base de datos sencilla. Puede usarse para representar catálogos de productos, listas de usuarios, bibliotecas, etc.
  • Representación de estructuras: Es ideal para representar estructuras anidadas (como menús, árboles genealógicos o estructuras organizacionales).

Características:

  • Auto-descriptivo: Las etiquetas describen el contenido.
  • Jerárquico: Crea una estructura en forma de árbol (elementos dentro de otros elementos).
  • Compatible con múltiples tecnologías: Usado en conjunto con XSL, XPath, XSLT, etc.
  • No tiene presentación visual: Solo estructura y datos. Para mostrarlo de forma visual, se combina con otras tecnologías (como XSL o CSS).
